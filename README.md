## Run Locally using Git

Clone the project

```bash
  git clone https://github.com/MoisalexWeb/extension-manager.git
```

Go to the project directory

```bash
  cd extension-manager
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
